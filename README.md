# Computer & Hardware E-Commerce Platform

This project is a full-stack e-commerce website for computer and hardware sales, inspired by Flipkart, with a professional UI/UX.

## Tech Stack
- **Frontend:** Next.js, TailwindCSS, shadcn/ui, Inter & Roboto fonts
- **Backend:** Express.js, MongoDB (via Mongoose)


## Features
- Product listing, search, filters
- Cart, checkout, user accounts, order history, wishlist
- Reviews, admin panel, profile, help & support
- Professional, modern layout and fonts
- Header with logo, search, account, cart

## Main Routes
- `/` — Home (product grid)
- `/cart` — Cart
- `/account` — Account/login
- `/orders` — Orders
- `/wishlist` — Wishlist
- `/admin` — Admin panel
- `/profile` — Profile
- `/checkout` — Checkout
- `/reviews` — Product reviews
- `/help` — Help & support

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```

## Customization
- UI components: shadcn/ui
- Fonts: Inter, Roboto (see src/app/globals.css)
- MongoDB connection: configure in backend with .env
